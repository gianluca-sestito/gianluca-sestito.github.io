## ğŸ“¦ Deployment to GitHub Pages

This repository is configured to automatically deploy to GitHub Pages when you push to the `main` branch.

### Automatic Deployment

1. Push your code to the `main` branch:
   ```bash
   git add .
   git commit -m "Update portfolio"
   git push origin main
   ```

2. GitHub Pages will automatically build and deploy your site
3. Your site will be available at `https://gianluca-sestito.github.io`

### Manual Setup (if needed)

1. Go to your repository settings on GitHub
2. Navigate to **Pages** in the left sidebar
3. Under **Source**, select `main` branch and `/ (root)` folder
4. Click **Save**
5. Your site will be live in a few minutes
